Before you run the script 'run.bat', you should do following steps:

1. build the 'java-project'.
2. put all the jar file to the 'lib' directory.
3. put BGM file to 'bgm' directory and dry sound file to the 'dry_sound' directory
4. edit dry_sound-bgm-list.xls file just as the sample in it. (please do not modify the first line, i.e. the 'src' and 'bgm' line)
5. double-click 'run.bat' run the synthesis scripts and enjoy.
